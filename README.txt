1) Download TUIO_NET-1.1.5
http://prdownloads.sourceforge.net/reactivision/TUIO11_NET-1.1.5.zip?download

2) Öffne das Projekt in VS (TUI_WEB.sln)

3) im Projektmappen-Explorer Rechtsklick auf Projektmappe "TUI Web"
4) hinzufügen -> vorhandenes Projekt -> TUIO_Lib ( aus Schritt 1 geladen)
5) rechtsklick auf Verweise->Verweis hinzufügen
6) Wählre Projektmappe und aktiviere TUIO_LIB

7) fertig
